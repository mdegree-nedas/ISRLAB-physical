1- Raspberry OS install (Using Linux):
	- Download Raspberry OS image from official site

	- mkdir image

	- unzip 2021-03-04-raspios-buster-armhf.zip -d image

	- cd image

	- sudo dd if=2021-03-04-raspios-buster-armhf.img of=/dev/mmcblk0 bs=1M conv=fsync status=progress
	(/dev/mmcblk0 is the location of the SDcard in which you will install the OS)


- (Now we focus on the Raspberry)


2- Docker install:
	- sudo apt update

	- sudo apt upgrade

	- sudo apt install -y apt-transport-https ca-certificates curl gnupg2 software-properties-common

	- curl -fsSL https://download.docker.com/linux/$(. /etc/os-release; echo "$ID")/gpg | sudo apt-key add -

	- echo "deb [arch=$(dpkg --print-architecture)] https://download.docker.com/linux/$(. /etc/os-release; echo "$ID") $(lsb_release -cs) stable" | sudo tee /etc/apt/sources.list.d/docker.list

	- sudo apt update

	- sudo apt install -y --no-install-recommends docker-ce cgroupfs-mount

	- sudo systemctl enable docker

	- sudo systemctl start docker

	- sudo usermod -aG docker pi

	- sudo systemctl reboot

	- Test installation:
		- docker run hello-world


3- Docker compose install:
	- sudo apt update

	- sudo apt install python3-pip libffi-dev

	- sudo pip3 install docker-compose


4- ROS2 Dashing full environment image creation:
	- docker pull ros:dashing

	- docker run -it ros:dashing
		- (follow ROS2 full environment install when you are inside the container)
		- (When you complete the install, exit the container without removing it)

	- docker login

	- docker commit <ros2_container_id> isrlab/ros2_dashing_rpi:full

	- docker push isrlab/ros2_dashing_rpi:full


- ROS2 full environment install (inside Docker container):
	- sudo apt update && sudo apt install locales

	- sudo locale-gen en_US en_US.UTF-8

	- sudo update-locale LC_ALL=en_US.UTF-8 LANG=en_US.UTF-8

	- export LANG=en_US.UTF-8

	- sudo apt update && sudo apt install curl gnupg2 lsb-release

	- sudo apt update

	- sudo apt install ros-dashing-desktop

	- source /opt/ros/dashing/setup.bash

	- sudo apt install -y python3-pip

	- pip3 install -U argcomplete

	- Test installation:
		- ros2 run demo_nodes_cpp talker

		- ros2 run demo_nodes_py listener
